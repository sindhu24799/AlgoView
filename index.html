Bubble Sort

Time Complexity: Best: O(n) | Average: O(n²) | Worst: O(n²)

Space Complexity: O(1) | Stable: Yes

Bubble Sort repeatedly compares adjacent elements and swaps them if they are in the wrong order. The largest elements "bubble" to the end with each pass.
